## Frontend Style Guide1

- React 视图组件使用 tsx，其它 reducer,interface 等文件使用 ts
- 路由级组件，class 名称增加 Page 后缀
- 命名上统一为小写中线命名法
- CSS 样式如果多单词，也采用中线命名法
- 其它规则由既定 3 lint 控制约束
